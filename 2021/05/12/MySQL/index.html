介绍

相关概念

  • DB：数据库，保存一组有组织的数据的容器
  • DBMS：数据库管理系统，用于管理DB中的数据
  • SQL：结构化查询语言，用于和DBMS通信的语言

登录

mysql [-h 主机名 -P 端口号] -u用户名 -p密码

MySQL常见命令

  1. 查看当前所有的数据库：show databases;
  2. 打开指定的库：use 库名;
  3. 查看当前库的所有表：show tables;
  4. 查看其它库的所有表：show tables from 库名;
  5. 创建表：create table 表名(列名 列类型,列名 列类型,…..);
  6. 查看表结构：desc 表名;
  7. 查看服务器版本：select version(); mysql – version mysql –V

MySQL语法规范

  1. 不区分大小写，但建议关键字的大写，表名，列名小写

  2. 每条命令最好用分号结尾

  3. 每条命令根据需要，可以进行缩进或换行

  4. 注释

单行函数

  • 字符函数

    • length：获取参数值的字节个数
    • concat：拼接字符串
    • upper，lower：大小写
    • substr、substring：注意索引从1开始
    • instr(str, substr)：返回后面的字符串在前面字符串第一次出现的索引，如果没有返回0
    • trim(‘a’ a45a)：将字符串左右的字符删掉
    • lpad(str, num, substr)：用指定字符实现左填充指定长度
    • rpad：右填充
    • replace(str, before, after)：替换

  • 数学函数

    • round：四舍五入
    • ceil：向上取整
    • floor：向下取整
    • truncate：截断，取几位
    • mod：取余

  • 日期函数

    • now：返回值当期系统日期+时间
    • curdate：返回当前系统日期，不包含时间
    • curtime：返回当前时间，不包含日期
    • str_to_date：字符串转换为日期
    • date_format：将日期转换为字符

  • 流程控制函数

    • if(10>5, ‘big’, ‘small’)：相当于三元运算符

    • case函数：

内连接：inner
  • 添加排序、分组、筛选
  • inner可以省略
  • 筛选条件可以放在where后面，连接条件放在on后面，提高分离性，便于阅读
外连接
  • 外连接的查询结果为主表中的所有记录，如果从表中有匹配的，显示匹配的值，从表中没有，则显示null
  • 左外连接，left jion左边的是主表，右外连接，right jion右边的是主表

子查询

含义

出现在其他语句中的select语句，成为子查询或内查询

外部的查询语句，成为主查询或外查询

分类

  • 按结果集的行列数不同

    • 标量子查询（一行一列）
    • 列子查询（多行一列）
    • 行子查询（一行多列）
    • 表子查询（多行多列）

  • 按子查询出现的位置

    • select后面

      • 标量子查询

    • from后面

      • 表子查询

    • where或having后面

      • 标量子查询

      • 列子查询（多行子查询）

        in, any\some—>min, all—>max

      • 行子查询

    • exisits后面（相关子查询）

      • 表子查询
